## C#

C# 实际上在 Windows 上就有一个叫做 csc 的编译器，在你的 C 盘 -> Windows -> Microsoft.NET -> Framework64(也有可能没有 64，具体根据你的系统) -> (版本号) -> csc.exe

C# 开发 IDE 建议使用 Visual Studio
